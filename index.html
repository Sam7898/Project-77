Visual Studio and Console Logs

Visual Studio Code is a coding platform where you can code in CSS, Java, and HTML. It is free to download, and user-friendly.

Console log shows you the errors in the website that you code, and is very helpful when coding with Visual Studio Code. You can open the Console by right-clicking on a website and clicking on inspect. Then click on the console tab on the right side. You can write anything you want in the console log as well.
